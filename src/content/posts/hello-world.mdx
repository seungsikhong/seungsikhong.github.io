# Hello World! 🎉

이것은 **MDX 시스템**의 첫 번째 테스트 포스트입니다.

## MDX의 장점

MDX를 사용하면 다음과 같은 장점이 있습니다:

- **마크다운**의 간편함
- **React 컴포넌트** 삽입 가능
- **프론트매터**로 메타데이터 관리
- **TypeScript** 지원

## 코드 예시

```typescript
function greet(name: string) {
  return `Hello, ${name}!`;
}

console.log(greet('MDX'));
```
